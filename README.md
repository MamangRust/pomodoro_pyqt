### Pomodoro Task QT



Pomodoro Task QT adalah aplikasi Pomodoro Timer dengan fitur Tasks List.



## Feature List
-   Tasks List dan Pomodoro Timer
-   Visualiasi Perbulan
-   Visualiasi Perminggu
